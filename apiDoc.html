Getting started

How it works

ShareHUB API uses a php page to get and add data from/to the DB. You can use it as a gateway to get the stored data from the HUB and add data to it.

How data stored

Data is stored in a MySQL database. When you use the API, it will query the DB and returns a JSON file you can use in your application.

Data types

On the website, if you add a new snippet, you can give it a name, a description, content and files (optional). With the API, you can only add text, no files. (yet.)

Requirements

To use the API, you have to have an account on ShareHUB, after that you need an API key to exchange data.

API Key

To get an API key, go to your profile details by clicking on your name on the profile page. Your API key will be below the used space.

Data exchange

To get/add data, you have to make a POST request. The details are below:

  • Address: http://blawg.hu/demo/sharehub/api.php?method=METHOD
  • Parameters: different for GET/ADD, listed below

Getting data

  • URL Parameters:
    • method=GET
  • POST Parameters:
    • apiKey: API_KEY

Adding data

  • URL Parameters:
    • method=ADD
  • POST Parameters:
    • snippetTitle: The title of the Snippet
    • snippetDescription: The description of the Snippet
    • snippetContent: The content of the Snippet

Sample requests

  • http://blawg.hu/demo/sharehub/api.php?method=GET
  • POST: apiKey=APIKEY
This will return with all the Snippets stored on the HUB associated with the API key in a JSON file format.
  • http://blawg.hu/demo/sharehub/api.php?method=ADD
  • POST: apiKey=APIKEY
  • POST: snippetTitle=TITLE
  • POST: snippetDescription=DESCRIPTION
  • POST: snippetContent=CONTENT
This will add a Snippet to the HUB associated with the API key.
